## 中文版

### 扩展名称
Side Stash

### 简短说明（<= 132 字）
右键保存网页中的文本、链接、图片到侧边栏，支持筛选与一键复制。

### 详细说明
Side Stash 是一个轻量的浏览器侧边栏收藏工具。浏览网页时，只需右键即可保存选中文本、链接或图片，并在侧边栏里集中查看、筛选、复制。

#### 核心功能
- 右键保存：支持文本、链接、图片
- 侧边栏列表：自动持久化
- 快速筛选：按类型和关键词/网址筛选
- 多选与一键复制：按行复制筛选结果
- 单条复制与删除

#### 使用方式
1. 在网页中选中文本或右键链接/图片
2. 选择“保存到侧边栏”
3. 打开侧边栏查看、筛选、复制或删除

### 权限说明
- `contextMenus`：提供右键菜单入口
- `storage`：本地持久化保存内容
- `tabs`：读取当前页面信息（标题/URL）
- `host_permissions: <all_urls>`：在任意网页启用右键保存

### 数据与隐私
- 所有数据仅保存在本地（`chrome.storage.local`）
- 不上传任何内容到远程服务器
- 不进行用户追踪或分析

### 隐私政策（可直接使用）
本扩展不会收集、传输或分享任何用户数据。所有保存内容仅存储在用户本地浏览器中。扩展不包含第三方分析或广告 SDK。

## English Version

### Extension Name
Side Stash

### Short Description (<= 132 chars)
Right-click to save text, links, or images to a side panel. Filter and copy instantly.

### Detailed Description
Side Stash is a lightweight side-panel collector for your browser. Save selected text, links, or images with a right-click, then browse, filter, copy, or delete items in one place.

#### Key Features
- Right-click save: text, links, images
- Side panel list with local persistence
- Quick filters by type and keyword/URL
- Multi-select and one-click copy (newline separated)
- Single-item copy and delete

#### How It Works
1. Select text or right-click a link/image
2. Choose “Save to side panel”
3. Open the side panel to view, filter, copy, or delete

### Permissions Rationale
- `contextMenus`: add right-click menu items
- `storage`: persist saved items locally
- `tabs`: read page title/URL for context
- `host_permissions: <all_urls>`: enable saving on any site

### Data & Privacy
- All data is stored locally in `chrome.storage.local`
- No data is sent to any server
- No tracking or analytics

### Privacy Policy (ready to use)
This extension does not collect, transmit, or share any user data. All saved content stays on the user’s local browser. The extension contains no third-party analytics or advertising SDKs.
